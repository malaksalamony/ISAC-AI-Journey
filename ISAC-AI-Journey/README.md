# Integrated Sensing and Communications (ISAC) AI Roadmap

This repository tracks my learning progression from fundamental data-driven models to Agentic AI, tailored specifically for wireless engineering and ISAC system design using MATLAB.

## Repository Structure

### [Phase 1: Foundational Machine Learning](./Phase1_Foundational_ML/)
- **Project 1: Path Loss Predictor:** Replaces traditional empirical formulas with a data-driven Regression Tree.
- **Project 2: Radar Target Filter:** Binary classification separating valid physical targets from environmental clutter.

### [Phase 2: Deep Learning](./Phase2_Deep_Learning/)
- **Project 3: Modulation Classifier:** Deep neural network classifying IQ signals.

### [Phase 3: Generative AI](./Phase3_Generative_AI/)
- **Project 4: Channel Generator:** Autoencoders creating synthetic channel state profiles.
- **Project 5: Local 3GPP Bot:** Interfacing text models to interpret standardization parameters.

### [Phase 4: Agentic AI](./Phase4_Agentic_AI/)
- **Project 6: Resource Allocation Agent:** Autonomous optimization looping through MATLAB execution blocks.
